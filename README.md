# ModuleSample

## Setup


Add this to your project build.gradle
``` gradle
allprojects {
    repositories {
        maven { url 'https://jitpack.io' }
    }
}
```
Add this to your module build.gradle

```gradle
   dependencies {
        implementation 'com.github.Droidhelios:ModuleSample:1.0'
    }

```
```
